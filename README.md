# 旅游推荐系统 python71

## 项目概述

旅游推荐系统是一个基于Python开发的Web应用，其功能包括用户系统、数据展示、用户互动、算法推荐、数据分析等。系统使用最新的旅游数据，通过机器学习算法为用户提供智能化的景点推荐。

## 技术栈

- 后端：Django
- 开发语言：Python
- 数据库：MySQL
- 爬虫技术：requests
- 推荐算法：机器学习
- 前端：BootStrap + ECharts

## 功能模块

### 用户系统

- 登录
- 注册
- 个人信息管理

### 数据展示

- 旅游景点信息可视化呈现

### 用户互动

- 景点评论
- 评分
- 反馈

### 算法推荐

- 基于机器学习智能推荐景点

### 数据分析

- 多维度统计旅游信息

### 可视化图表

- ECharts 图表展示

## 系统角色

- 用户：浏览信息、发表评论、评分、查看推荐景点
- 管理员：管理用户、维护数据、监控系统运行状态

## 运行环境

- Python 3.x
- Django 2.x
- MySQL 5.x
- 兼容主流浏览器
